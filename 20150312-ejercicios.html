Ejercicios HTML-PHP

Cada uno de los siguientes ejercicios es independiente, es decir, por cada ejercicio debes crear un archivo .php. El desarrollo y prueba de las páginas .php se hacen usando el servidor WAMP.

Los ejercicios se deben entregar en engrade.com en Parcial 2 en la correspondiente Actividad.

  1. [engrade: Ejercicio TABLE HTML PHP] Usando PHP, generar tabla HTML con 100 renglones y dos columnas. Puedes imprimir de forma directa los tags HTML o usando echo dentro de bloques de código php. Para crear una tabla con una celda se usa (tr=renglon, td=celda):

                    <table><tr><td>valor</td></tr></table>
    • En la primera columna debes imprimir los numeros del 100 a 1, uno en cada renglon.
    • En la segunda columna debes imprimir los numeros de 1 al 100, uno en cada renglon.

    TIP: usa for o while para generar la tabla y la numeración. Con un solo loop puedes resolver el ejercicio.

  2. [engrade: Ejercicio operador MODULO PHP] Usando el operador %(módulo) de PHP:

    • crear formulario (<form>) HTML con un campo <input> donde el usuario va a capturar un numero entero
    • recuperar el valor capturado por el usuario usando la misma pagina donde creaste el <form>
    • imprimir o indicar al usuario si el número que capturó es PAR o IMPAR

  3. [engrade: Ejercicio TABLE dinamico] Usando PHP, generar una tabla HTML con n renglones y m columnas, donde n y m son datos capturados por el usuario en un formulario web. Colocar en cada celda de la tabla el texto "YOLO".

  4. [engrade: Ejercicio FUNCTION] Usando el código del ejercicio anterior, generar una tabla HTML con n renglones y m columnas, donde n y m son datos capturados por el usuario en un formulario web. Colocar en cada celda de la tabla el texto "YOLO". La diferencia ahora es que la tabla se debe imprimir o generar usando una funcion (que se llame imprimirTabla()), por lo cual la funcion debe recibir 3 argumentos: uno es el numero de renglones, otro el numero de columnas y el tercer el texto a imprimir en las celdas.
